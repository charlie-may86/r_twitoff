# r_twitoff
A second version of twitoff, but following Mike R.

## To install

Download the repo and navigate there from the command line

```sh
git clone <address>
cd into r_twitoff
```

## setup

Set and up activate a virtual enviroment

```sh
pipenv install 
pipenv shell
```

## To run web_app
```sh
FLASK_APP=web_app flask run
```
